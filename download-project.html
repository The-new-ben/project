🏛️ HyperCourt Enterprise - הורדת פרויקט

מערכת משפטית מתקדמת עם ארכיטקטורה מיקרו-שירותים

🚀 הוראות התקנה והפעלה:

  1. חלץ את הקובץ למיקום הרצוי במחשב
  2. פתח טרמינל בתיקיית הפרויקט
  3. התקן תלויות: npm install
  4. הפעל את האפליקציה: npm run dev
  5. פתח בדפדפן: http://localhost:5173

🔧 תכונות עיקריות:

  • 🤖 בינה מלאכותית: ניתוח משפטי מתקדם
  • ⚖️ מערכות משפט: Common Law, Civil Law, הלכה יהודית ועוד
  • 🔍 חיפוש מתקדם: חיפוש והדגשה בדיונים
  • 💾 שמירה מקומית: IndexedDB לשמירת נתונים
  • 📊 פאנל ניהול: סטטיסטיקות וגיבוי נתונים
  • 🎨 עיצוב מתקדם: Tailwind CSS ואנימציות
  • 🔒 אבטחה: JWT, Rate Limiting, Validation

🌐 פריסה לייצור:

  1. בנה את הפרויקט: npm run build
  2. העלה לשרת: תיקיית dist/
  3. הגדר משתני סביבה בשרת הייצור
